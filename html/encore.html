Installation

Copy the following css link in the head tag of your HTML document and the JS link before the closing body tag and you are good to go. 


            
            <link rel="stylesheet" href="https://encoreui.netlify.app/components/index.css />
            
            <script src="https://encoreui.netlify.app/script.js"></script>
